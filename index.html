Projects

Robotic Emotion Modeling

Bio-inspired emotions from microbial fuel cells.

Designed and built a microbial fuel cell system to simulate emotional states in robots.

Mapped electrical output to a circumplex emotion model enabling dynamic personality modulation.

Tools: Microbial Fuel Cells, Embedded Systems

Quadruped Locomotion Prototype

PPO training for stable robotic gait.

Built the URDF model of a quadruped and integrated it into PyBullet for rhythmic locomotion training.

Modified PPO reward functions to improve gait stability and convergence, achieving ~75% average reward.

Tools: URDF, PyBullet, PPO, Python

BotTube Autonomous Pipeline

Agentic pipeline creating daily videos.

Autonomous system using LangChain, GPT-4 and the YouTube API to collect data, generate scripts and visuals, and upload ~3 videos per day.

Tools: LangChain, GPT-4, YouTube API

Vision-Guided Teleoperation

Hand tracking controlling InMoov arm.

Built and assembled an ESP32-controlled InMoov arm with custom PID control and Bluetooth interface.

Used MediaPipe and vector math to extract finger angles and compute real-time joint errors for teleoperation.

Tools: ESP32, MediaPipe, PID Control
