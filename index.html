Architecture Decisions

Teaching Drones to
Coordinate Without a
Central Controller

Each drone: own state + neighbour observations → independent policy → residual thrust + moment output
Policy Type
Independent per agent · no central controller. Centralised control is a single point of failure. Doesn't scale to a port fleet.
State Space
Own state + neighbour observations. Coordination emerges from training · not hard-coded rules that break on edge cases.
Action Space
Residual thrust + moment over a structured baseline. Better training stability. Maps cleanly to real actuator constraints.
Training Condition
Crate mass and size randomised throughout training. Variable payloads are the training condition · not an edge case the policy sees once.
